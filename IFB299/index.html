From the drop-down menu, select the encoding with which you created the file. Important: Selecting the wrong initial encoding may corrupt the file.

If you’re not sure of the correct encoding, here are some tips:
  • If you are using a Western European language like English, Spanish, or French without any special characters, select “ISO-8859-1”.
  • If you are using a non-Western language, or you have special characters that are not HTML-encoded in your file, select “utf-8”.
  • You may have selected the wrong encoding if your file appears with text like this: If this happens, immediately abort the edit and select the correct encoding. Do not save the file; this may corrupt the text permanently.
